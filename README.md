# my-pixel-pet

Digital Tamagotchi-style pixel pet built with **React + Vite**.

## Quick start

```bash
npm install
npm run dev
```

Open:
- http://localhost:5173/my-pixel-pet/

## Production build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Live site:
- https://alexsgittyhub.github.io/my-pixel-pet/

## Verify you are on the latest deployed build

Inside the app header, look for the badge:
- `BUILD v1.4.1`

If the site looks stale after deploy:
1. hard refresh (`Ctrl/Cmd + Shift + R`)
2. open in an incognito/private tab
3. ensure you are visiting the project path with trailing slash:
   - `https://alexsgittyhub.github.io/my-pixel-pet/`
